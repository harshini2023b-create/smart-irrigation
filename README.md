# Smart Irrigation System 🌱💧

## 📌 Overview
This project is an intelligent irrigation system that uses machine learning to automate watering decisions based on environmental conditions.

## 🚀 Features
- Predicts irrigation needs
- Uses trained ML model (.pkl files)
- Efficient water usage
- Flask-based application

## 🛠️ Tech Stack
- Python
- Flask
- Machine Learning (Scikit-learn)
- Pandas, NumPy

## 📂 Project Structure
- app.py → Main application
- train_model.py → Model training
- *.pkl → Trained models
- pages/ → UI pages

## 🎯 Outcome
Optimizes water usage and improves crop efficiency.
